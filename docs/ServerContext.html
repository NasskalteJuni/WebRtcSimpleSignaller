Class: ServerContext

ServerContext(context)

A Context Object produced by the Hub that handles message context. It is intended to be based on a received message on the server and offers functionality to handle the received message accordingly. This can be a reply to the given message, offering the user that sent the message as property and other utility functions, like allowing a call to join(channelId) to let the user join the given Channel.
