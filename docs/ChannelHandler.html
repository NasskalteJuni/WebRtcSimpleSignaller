Class: ChannelHandler

ChannelHandler()

A Channel is a set of 1 or more Members, that can send and receive direct and broadcast messages between each other. The Channel takes care of Member count, etc. by sending and receiving control messages to the server, that stores this state. The Listenable-Mixin is used to trigger on('type', (content, msg) => ...) event handlers for messages that are sent by other clients or for updates to the members of the channel
